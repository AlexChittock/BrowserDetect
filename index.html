BrowserDetect

BrowserDetect is a simple Flash and JS pairing that allows deeper probing of the client's browser and OS beyond what's normally available in JS.

What's available

  • Operating system - GetOS()
  • System language - GetLang()
  • Screen resolution - GetRes()
  • Screen colours - GetColour()

Getting started

  1. Download the detection script at: https://github.com/AlexChittock/BrowserDetect/downloads.
  2. Include the flash file on your pages.
  3. Handle the detected information in a function called OSDetectReady. For example:
function OSDetectReady()
{
    var obj = swfobject.getObjectById("flash-detect-placeholder");
    if (obj) {
        $("input[name=OperatingSystem]").val(obj.GetOS());
        $("input[name=Resolution]").val(obj.GetRes());
        $("input[name=ScreenDepth]").val(obj.GetColour());
    }
}
